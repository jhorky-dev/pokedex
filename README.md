## Pokedex

## Instalación

```bash
npm install
```

## Uso

```bash
npm start
```

## Desarrollo

### Desarrollo Frontend

```bash
npm run dev
```

### Desarrollo Backend

```bash
npm run dev:server
```

### Desarrollo Backend y Frontend

```bash
npm run dev:all
```

## Despliegue

```bash
npm run build
```

```bash
npm run deploy
```
